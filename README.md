# ScaleScraper

This project Web-scrape articles from the URL

## Install & Run
Run `npm i` and `npm run build` for setup

Run `npm start` for starting whole app and nodejs server

Open http://localhost:4200/ in browser

Optimized for scraping articles from https://www.bbc.com/news/uk

## Known bugs
Not all sections of https://www.bbc.com/ because of major differences in markup between
 different sections
